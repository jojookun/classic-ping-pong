# 🎮 NEON PONG

A retro-style, neon-themed Ping Pong game built with pure HTML, CSS, and JavaScript. Play against an AI or challenge a friend in local multiplayer!

## 🕹️ Play Online

👉 **[Click here to play!](https://jojookun.github.io/classic-ping-pong/)**

No installation needed — just open the link and play in your browser.

---

## 🎯 Game Features

- **VS AI Mode** — Fight against the computer with 5 difficulty levels: Easy, Medium, Hard, Insane, Extreme
- **VS Player Mode** — Local 2-player mode on the same keyboard
- **Custom Names** — Enter your own player names before the match
- **Speed Settings** — Choose from Slow (0.5x), Normal (1x), Fast (1.5x), or Hyper (2x)
- **Themes** — Switch between Matrix Green, Cyber Magenta, Tron Cyan, and Retro White
- **Goal Score** — Set a custom win score (3–10 goals)

---

## ⌨️ Controls

| Action         | Player 1 | Player 2       |
|----------------|----------|----------------|
| Move Up        | `W`      | `↑` (Up Arrow) |
| Move Down      | `S`      | `↓` (Down Arrow)|

## 🚀 How to Run Locally (Optional)

If you want to run the game on your own computer:

1. **Download or clone this repository:**
   ```bash
   git clone https://github.com/jojookun/classic-ping-pong.git
   ```

2. **Open the game:**
   - Navigate to the downloaded folder.
   - Double-click `index.html` to open it in your browser.

3. That's it — no server, no install required! ✅

---

## 🌐 How to Deploy to GitHub Pages (for the owner)

1. Go to your repository on GitHub: `https://github.com/jojookun/classic-ping-pong`
2. Click **Settings** → **Pages** (left sidebar)
3. Under **"Branch"**, select `main` and set folder to `/ (root)`
4. Click **Save**
5. Wait ~1 minute, then your game will be live at:
   `https://jojookun.github.io/classic-ping-pong/`
